# Nasdaq (nasdaq-omx-group)

Nasdaq is a global technology company serving capital markets and other industries, providing trading, clearing, exchange technology, listing, information, and public company services. Nasdaq Data Link offers REST APIs for accessing financial, economic, and alternative data.

## APIs

### Nasdaq Data Link Time-series API
REST API for retrieving time-series financial and economic data including dataset values, metadata, and combined responses.
- **Base URL:** `https://data.nasdaq.com/api/v3`
- [Documentation](https://docs.data.nasdaq.com/docs/in-depth-usage) | [GettingStarted](https://docs.data.nasdaq.com/docs) | [SignUp](https://data.nasdaq.com/sign-up)

### Nasdaq Data Link Tables API
REST API for retrieving Tables-style datasets with filtering and bulk download support.
- **Base URL:** `https://data.nasdaq.com/api/v3`
- [Documentation](https://docs.data.nasdaq.com/docs/tables-1) | [SignUp](https://data.nasdaq.com/sign-up)

### Nasdaq Data Link Streaming API
Real-time market data delivery via persistent streaming connections.
- **Base URL:** `https://data.nasdaq.com/api/v3`
- [Documentation](https://docs.data.nasdaq.com/docs) | [SignUp](https://data.nasdaq.com/sign-up)
